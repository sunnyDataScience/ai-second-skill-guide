# AI Second Skill Guide (2026)

A practical, workflow-first guide to help professionals build an AI-powered second skill:
- 4 learning paths
- Task-to-tool matrix (no hype, just outcomes)
- 30-day project plan (checklist style)
- Monetization strategies and portfolio packaging

## Repository Structure
- `docs/ebook/` -- outline, manuscript, assets, PDF exports
- `docs/landing/` -- landing page copy and section drafts
- `docs/templates/` -- reusable templates (STRIKE, matrices, worksheets)

## Versioning
This project follows Semantic Versioning (MAJOR.MINOR.PATCH).
See `VERSION` and `CHANGELOG.md`.

## Release Process
1. Update manuscript and templates
2. Update `VERSION` and `CHANGELOG.md`
3. Export PDF to `docs/ebook/exports/`
4. Create a git tag `vX.Y.Z` and publish a GitHub Release
